## 3. Meeting Objectives

**Success Definition:** CTO gives verbal approval to proceed with a 6-week latency benchmark POC on AWS; Chief Architect agrees to participate in POC design.

**Strategic Alignment:** This engagement supports AWS's goal of becoming the primary cloud for APAC exchange operators (alongside SGX, ASX, JPX wins).

### Objective 1: Demonstrate AWS Low-Latency Trading Capabilities

| Item | Detail |
|------|--------|
| **Desired Outcome** | Customer confidence that AWS can meet <50μs jitter requirements for order matching |
| **Context** | HKEX experienced latency inconsistency on Azure during peak hours; this is their #1 pain point |
| **Talking Points** | Nitro architecture, dedicated hosts, placement groups, kernel bypass networking, Graviton4 benchmarks from SGX |
| **Ask** | Agreement to run comparative latency benchmarks in POC |
